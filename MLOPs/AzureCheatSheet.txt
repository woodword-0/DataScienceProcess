Azure Account:

az login: Log in to your Azure account.
az account show: Show the details of the current Azure account.
az account list: List all Azure subscriptions.
Resource Groups:

az group create --name <group_name> --location <location>: Create a new resource group.
az group delete --name <group_name>: Delete a resource group.
az group list: List all resource groups in the current subscription.
Virtual Machines:

az vm create --resource-group <group_name> --name <vm_name> --image <image> --admin-username <username> --admin-password <password>: Create a new virtual machine.
az vm start --resource-group <group_name> --name <vm_name>: Start a virtual machine.
az vm stop --resource-group <group_name> --name <vm_name>: Stop a virtual machine.
az vm list: List all virtual machines in a resource group or subscription.
App Services:

az appservice plan create --name <plan_name> --resource-group <group_name> --sku <sku>: Create an app service plan.
az webapp create --name <app_name> --plan <plan_name> --resource-group <group_name> --runtime <runtime>: Create a web app.
az webapp delete --name <app_name> --resource-group <group_name>: Delete a web app.
az webapp list: List all web apps in a resource group or subscription.
Storage Accounts:

az storage account create --name <account_name> --resource-group <group_name> --location <location> --sku <sku>: Create a storage account.
az storage account show --name <account_name> --resource-group <group_name>: Show the details of a storage account.
az storage account list: List all storage accounts in a resource group or subscription.
Networking:

az network vnet create --name <vnet_name> --resource-group <group_name> --location <location> --address-prefix <address_prefix>: Create a virtual network.
az network nsg create --name <nsg_name> --resource-group <group_name> --location <location>: Create a network security group.
az network lb create --name <lb_name> --resource-group <group_name> --frontend-ip-name <ip_name> --backend-pool-name <pool_name>: Create a load balancer.




Create a Service Principal:

az ad sp create-for-rbac --name <app_name>: Create a Service Principal with a randomly generated password.
az ad sp create-for-rbac --name <app_name> --password <password>: Create a Service Principal with a specific password.
az ad sp create-for-rbac --name <app_name> --cert <certificate_file>: Create a Service Principal using a certificate.
List Service Principals:

az ad sp list: List all Service Principals in the current Azure subscription.
Assign Roles to Service Principal:

az role assignment create --assignee <sp_id> --role <role_name>: Assign a built-in role to the Service Principal.
az role assignment create --assignee <sp_id> --role <role_name> --scope <scope>: Assign a role to the Service Principal at a specific scope.
Delete a Service Principal:

az ad sp delete --id <sp_id>: Delete a Service Principal.
Get Information about a Service Principal:

az ad sp show --id <sp_id>: Get information about a Service Principal.
Update a Service Principal:

az ad sp credential reset --name <app_name>: Reset the password of a Service Principal.
az ad sp credential delete --id <sp_id> --password <password>: Delete a password credential for a Service Principal.
az ad sp credential reset --id <sp_id> --cert <certificate_file>: Reset the certificate of a Service Principal.
Manage Service Principal Secrets:

az ad app credential reset --id <app_id>: Reset the password or certificate of a Service Principal.
Remember to replace <app_name>, <password>, <certificate_file>, <sp_id>, <role_name>, and <scope> with the actual values specific to your use case.
